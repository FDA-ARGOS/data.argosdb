# data_files

This folder contains archived and reference data tables used throughout the project. These files include materials from Release 1.0 (December 2025), older v1.4 data table files, and additional historical datasets preserved for reproducibility and reference.

## Contents
- _Release 1.0 Tables (Dec 2025)_: Data tables corresponding to the official Release 1.0 of the dataset.
- _v1.4 Data Tables (Archived)_: Older versions of data tables maintained for version tracking and historical comparison.
- _HumanPathogens_Comprehensive.xlsx_: A curated reference table of human pathogenic organisms, including bacteria, viruses, and eukaryotic pathogens. This dataset integrates multiple sources and provides taxonomic identifiers and classification information for downstream analysis and database integration.

The current version of _HumanPathogens_Comprehensive.xlsx_ includes:
- 1,705 bacterial species
- 1,598 viral species
- 49 fungal and other eukaryotic organisms

## Notes
- Files in this directory, except for HumanPathogens_Comprehensive.xlsx, may not reflect the most current data schema or processing pipelines. Up to date data tablesw ould be on data.argosdb.org.
- These datasets are retained for reference, validation, and reproducibility purposes.
- For the most up-to-date data tables and workflows, refer to the main ARGOS project [README](https://github.com/FDA-ARGOS/data.argosdb/blob/main/README.md).
