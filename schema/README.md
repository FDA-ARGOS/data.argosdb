# ARGOS DB Schema versions
The schema directory contains JSON schema files that define the structure, required fields, and data types for ARGOS data tables.

These schemas are organized by version and data type, and are used to ensure consistency and standardization across all datasets.

## Notes
- Each subdirectory corresponds to a specific schema version or category (e.g., core, non-core)
- Schemas are used for validation of data tables and to enforce controlled vocabulary and formatting
- Users should reference the appropriate schema version when generating or validating data files
